Git & GitOps

Git is a distributed version control system widely used in software development to track changes in source code during software development. GitOps is a methodology that leverages Git pull requests to manage infrastructure and deployments.

Engineering Toolkit (ETK)

In Engineering Toolkit, we learned tools such as Visual Studio Code for software development and API testing tools for ensuring the functionality and reliability of our applications.
